# sprintsight-sandbox

Throwaway demo repo for Sprintsight's cross-tool watermelon (Goal B) live-verify.
Branches and PRs here deliberately reference real SSSB Jira keys to create a
known cross-tool watermelon scenario:

- SSSB-1 (Atlas): In Progress in Jira, no code work here -> watermelon
- SSSB-2 (Boreas): Done in Jira, PR left open/unmerged here -> watermelon
- SSSB-3 (Boreas): Done in Jira, PR merged here -> honest (not flagged)
